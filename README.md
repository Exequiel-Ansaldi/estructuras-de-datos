# estructuras-de-datos
Data Structures

Collection of implementations developed during the Data Structures course.

The objective of this repository is to practice fundamental computer science concepts and problem solving using Python.

## Topics covered

* Linked Lists
* Doubly Linked Lists
* Stacks and Queues
* Recursion
* Trees
* Graphs
* Hash mapping
* CSV data processing
* Object-Oriented Programming

## Purpose

Educational repository focused on understanding how data structures work internally rather than using built-in language utilities.

## Notes

Some exercises may contain alternative implementations exploring different approaches to the same problem.
